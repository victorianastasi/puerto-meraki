# Puerto Meraki

Proyecto ecommerce, dedicado a la venta de cuadros decorativos, construido con Create React App y Firebase.

## Demo - Puerto Meraki

![Puerto Meraki Demo](demo/puerto-meraki.gif)

### Dependencias
- React (v 17.0.2)
- Firebase (v 8.0.2)
- react-spinners (v 0.11.0)
- react-icons (v 4.2.0)

### UI 
- Bootstrap (v 4.4.1)
- Google Fonts
- Animate.css (v 4.1.1)

### Install
npm install

### Usage
npm start
